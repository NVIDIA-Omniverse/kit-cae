# Templates

These are application templates from [Kit Application Template](https://github.com/NVIDIA-Omniverse/kit-app-template)(KAT)
The directory includes templates corresponding to the latest/default version of KAT. These are used by
`repo.sh|bat template new` command as templates to add new extensions. The versioned subdirectories include `templates/app`
specific to a version of Kit. These are used automatically as the `*.base.kit` for applications when a specific Kit version
is chosen using `repo.sh|bat select_kit_version ...`.
