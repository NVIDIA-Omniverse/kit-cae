# CAE OpenFOAM Data

This extension provides a data delegate for reading OpenFOAM field arrays in Kit-CAE.

## Features

- Read OpenFOAM mesh data (points, faces, owner, neighbour)
- Read OpenFOAM internal fields
- Read OpenFOAM boundary fields
- Support for both binary and ASCII formats
- Support for partitioned (parallel) OpenFOAM cases
