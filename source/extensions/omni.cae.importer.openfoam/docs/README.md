# CAE OpenFOAM Asset Importer

This extension provides an asset importer for OpenFOAM files (.foam) in Kit-CAE.

## Features

- Import OpenFOAM case files
- Support for parallel (decomposed) cases
- Automatic detection and import of time steps
- Creation of boundary meshes
- Support for internal fields

## Usage

Use the Asset Importer to import .foam files. The importer will automatically detect the case structure and create the appropriate USD representation.
